Mac OS 版 Battle.Net 安装（战网客户端）
2016-02-22|Mac OS

战网安装错误

如果你按照普通的安装思路：

  1. 打开官网下载的压缩包
  2. 把里面的Battle.net拖出来，拖到你的目录
  3. 打开目录找到战网图标，会发现出现一个提示：“无法打开Battle.net-Setup”
  4. 普通思想走不了。接下来我们排查一下。

错误排查

来试试到底是发生了什么导致这个东西无法安装

普通错误#1：未设置Mac软件源许可

  1. 打开系统偏好设定
  2. 打开安全性与隐私
  3. 解锁并把软件许可设定为“任何来源”。
  4. 再次打开战网安装，依然提示：“无法打开Battle.net-Setup”

普通错误#2：编码错误

  1. 强制打开法，对战网安装包右键，选择“显示包内容”
  2. 打开以下路径 /Battle.net-Setup.app/Contents/MacOS/，打开最后这个Battle.net Setup
  3. 提示编码错误，安装包使用非UTF-8

权限错误

既然上面的方法都不可用，大概就是权限问题导致的了
