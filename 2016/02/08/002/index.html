插件使用

这里包含了装载的所有插件的使用方法以及配置方法。

首要插件 - Hexo Admin (hexo-admin插件)

在之前的建站教程中会有两个本地管理插件安装，这个是我认识的最早的插件，但是自定义的选项比较少。
这个插件的好与不好：

  1. 简洁明了
  2. 渲染效果不错
  3. 可以设置自己的密码来访问（我觉得Hexo Hey的界面更好看）
  4. 似乎是官方的
  5. 不能自定义显示字体（我还没找到修改的地方。我偏好中文选择黑体或者雅黑，但是原版是宋体…简直就是虐杀）
  6. 没有中文（可以汉化的说）

首要插件 - Hexo Hey (hexo-hey插件)

这是另一个本地管理插件，后面在Hexo官网上找到的，用起来感觉还不错。
这个插件的好与不好：

  1. 简洁明了（和Hexo Admin没什么区别）
  2. 渲染好看（渲染出来的是浏览器默认字体）
  3. 支持中文（懒人们可以不用汉化）
  4. 自定义选项很多，可以自己编辑
  5. 可以设置密码来访问
  6. 还是不能自定义字体2333333
  7. 界面的元素有点错位（可以自己改）
